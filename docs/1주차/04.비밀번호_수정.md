## 요구사항

- 필요 정보: 기존 비밀번호, 새 비밀번호
- 비밀번호 규칙을 따름
  1. 8~16자의 영문 대소문자, 숫자, 특수문자만 가능
  2. 생년월일은 비밀번호 내에 포함될 수 없음
- 현재 비밀번호와 동일한 비밀번호는 사용 불가
- 인증: `X-Loopers-LoginId`, `X-Loopers-LoginPw` 헤더 사용
