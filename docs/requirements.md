# Requirements

## User (회원)

### 회원가입

입력 정보: 로그인 ID, 비밀번호, 이름, 생년월일, 이메일

- [ ] 로그인 ID는 영문과 숫자만 허용한다
- [ ] 이메일은 `xx@yy.zz` 형식이어야 한다
- [ ] 생년월일은 `yyyy-MM-dd` 형식이어야 한다
- [ ] 이름은 빈 값일 수 없다
- [ ] 비밀번호는 8~16자의 영문 대소문자, 숫자, 특수문자만 허용한다
- [ ] 비밀번호에 생년월일을 포함할 수 없다
- [ ] 비밀번호는 암호화하여 저장한다
- [ ] 이미 가입된 로그인 ID로는 가입할 수 없다

---

### 내 정보 조회

인증 헤더: `X-Loopers-LoginId`, `X-Loopers-LoginPw`

반환 정보: 로그인 ID, 이름, 생년월일, 이메일

- [ ] 이름의 마지막 글자는 `*`로 마스킹한다
- [ ] 인증 헤더가 없으면 `401 Unauthorized`를 반환한다
- [ ] 존재하지 않는 로그인 ID로 조회하면 `404 Not Found`를 반환한다

---

### 비밀번호 수정

입력 정보: 기존 비밀번호, 새 비밀번호

- [ ] 기존 비밀번호가 일치해야 수정할 수 있다
- [ ] 새 비밀번호는 비밀번호 규칙을 따라야 한다
- [ ] 현재 비밀번호와 동일한 비밀번호로는 변경할 수 없다
- [ ] 인증 헤더가 없으면 `401 Unauthorized`를 반환한다
