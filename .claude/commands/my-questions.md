현재 대화에서 사용자가 했던 질문들을 정리해주세요.

## 규칙
1. 사용자가 직접 작성한 메시지에서 질문에 해당하는 내용만 추출합니다.
2. 단순 지시나 요청은 제외하고, 명시적으로 정보를 묻거나 의견을 구하는 질문만 포함합니다.
3. 질문은 대화 순서(시간순)대로 번호를 매겨 나열합니다.
4. 원문 그대로 유지하되, 맥락이 불분명한 경우 괄호 안에 간단한 보충 설명을 추가합니다.
5. 사용자 질문이 구체적이지 않은 경우 어떤 질문을 한건지 사용자에게 질문의 의도를 물어봐서 구체적으로 정리합니다.

## 출력 형식
**💬 사용자 질문 목록 (총 N개)**

1. [질문 원문] (필요시 보충 설명)
2. [질문 원문]
3. ...
