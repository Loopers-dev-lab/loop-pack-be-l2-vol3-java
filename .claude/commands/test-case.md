# 테스트 케이스 템플릿 생성

사용자가 입력한 테스트 메서드명으로 테스트 클래스에 테스트 케이스 껍데기를 작성합니다.

## 입력
- 테스트 메서드명 (한글 가능)

## 출력 형식
```java
@Test
void {메서드명}() {
    //given
    
    //when
    
    //then    
}
```

## 규칙
1. 메서드명은 사용자가 입력한 그대로 사용 (공백은 _로 변환)
2. @Test 어노테이션 포함
3. given/when/then 주석 포함
4. 중괄호 안은 비워둠
5. 현재 사용자가 IDE에서 열고 있는 테스트 파일에 작성
6. 마지막 메서드 뒤, 클래스 닫는 중괄호 앞에 추가

## 예시

입력: 중복_가입된_ID_입력시_예외_발생

출력: 
```java
@Test
void 중복_가입된_ID_입력시_예외_발생() {
    //given

    //when

    //then
}
```

## 사용법
/test-case 중복_가입된_ID_입력시_예외_발생
